# AI 架構圖索引

此資料夾將收錄與 AI 系統、模型訓練／推論流程相關的架構設計。若新增圖檔，請依照下列指引更新此檔案。

## 當前圖面
- 目前尚未收錄圖檔，歡迎新增並於此處補上 Markdown 預覽。

## 新增圖面指南
1. 將圖檔置於本資料夾，採 `{領域}-{主題}-{版本}.{副檔名}` 命名，例如 `ai-mlops-pipeline-v1.drawio`。
2. 在下方列表中加入條目，包含：
   - 簡短主題說明與使用情境。
   - Markdown 預覽：`![圖說](./檔名)`。
   - 若有補充文件（如決策紀錄、假設），可連結於同一段落。
